Executive SummaryZOL adalah game strategi Rock-Paper-Scissors (RPS) masif on-chain di Solana. Game ini menggunakan model "Lossless PvP", di mana pemain bertarung memperebutkan Yield (bunga) dari total kolam dana (Pool), tanpa risiko kehilangan modal utama.Peran x402: Protokol x402 bertindak sebagai "The Loot Engine" (Mesin Distribusi Hadiah). x402 mengotomatisasi penyelesaian permainan (settlement) dan mengubah raw yield menjadi aset game atau likuiditas yang langsung dikirim ke dompet pemenang tanpa proses klaim manual yang rumit.Core Loop: Deposit -> Choose Faction -> Battle (Epoch) -> x402 Payout.2. Definisi Masalah (GameFi Context)High Barrier to Entry: Game Web3 biasanya memaksa pemain membeli NFT mahal atau Token volatil di awal.Manual Grinding: Pemain harus melakukan "Claim Reward" manual setiap hari, membuang gas fee dan waktu.Token Inflation: Hadiah biasanya berupa token native game yang dicetak terus menerus (inflasi), sehingga harganya cenderung jatuh ke nol.3. Solusi ZOL: The Lossless Battle ArenaZOL menawarkan arena pertempuran di mana modal (USDC/SOL) aman, namun potensi keuntungan (Yield) diperebutkan secara kompetitif.Konsep Utama:Pemain memasukkan pasukan (Deposit Aset) ke salah satu Faksi.Aset tersebut menghasilkan Yield di backend (via protokol lending Solana).Di akhir putaran (Epoch), Faksi Pemenang mengambil Yield dari Faksi Kalah.x402 mengeksekusi transfer kemenangan tersebut secara instan.4. Rumus Kemenangan (The Yield War)Pemenang ditentukan oleh Selisih Dominasi terhadap Total Value Locked (TVL).Logika: Faksi mendapat poin dengan memangsa targetnya, tapi dikurangi poin jika dimangsa predatornya.Rumus Score: Score = (% TVL Target) - (% TVL Predator)Contoh Skenario:Jika Pool Vanguard (Batu) terlalu gemuk/banyak orang -> Maka Mage (Kertas) akan panen besar (High Yield).Pemain harus menebak: "Ke mana kerumunan (crowd) akan pergi minggu ini?" dan memilih faksi counter-nya.5. Integrasi x402: The Settlement LayerDi sinilah ZOL berbeda dari game DeFi biasa. x402 bukan sekadar fitur tambahan, tapi infrastruktur backend yang menangani arus kas game.Fungsi x402 dalam ZOL:1. Smart Payout (Distribusi Otomatis)Tanpa x402, pemain harus klik "Claim" dan bayar gas.Dengan x402, saat Epoch berakhir:Kontrak menghitung pemenang.x402 Trigger: Mengambil yield kemenangan dari pool kalah.x402 Action: Langsung "menyuntikkan" yield tersebut ke saldo deposit pemenang (Auto-Compounding) ATAU mengirimkannya ke wallet pemain sebagai profit bersih.2. Yield-to-Asset Conversion (In-Game Economy)Pemain bisa mengatur preferensi di x402 untuk mengubah yield kemenangan menjadi item game.Setting: "Jika saya menang, belikan saya 'Shield Potion' (NFT) untuk ronde berikutnya."x402 Execution: Menukar Yield USDC -> Membeli NFT Marketplace -> Mengirim NFT ke Inventory pemain. Semua dalam satu transaksi atomik di akhir Epoch.6. Arsitektur Teknis (Solana)Sistem ZOL dibangun di atas kecepatan Solana dengan struktur sebagai berikut:ZOL Vault (Program A):Menampung deposit pemain.Terhubung ke Yield Source (misal: Kamino/Marginfi) untuk generate bunga dasar.Menjaga rasio 1:1 Principal agar aset selalu aman.Game Logic (Program B):Menyimpan state Faksi (Batu/Gunting/Kertas).Menghitung Score di akhir Epoch.x402 Payment Adapter (Program C):Menerima sinyal pemenang dari Program B.Menarik Yield dari Program A.Mendistribusikan Yield sesuai instruksi pemain (Re-stake / Payout / Buy Item).7. Keunggulan Kompetitif (Why ZOL Works?)Pure Strategy, No Gambling:Karena pokok dana dikembalikan 100%, pemain tidak takut kalah. Ini menciptakan retensi pemain yang sangat tinggi (Sticky User).Zero-Friction Rewards (via x402):Pemain merasakan pengalaman "Seamless". Menang = Saldo bertambah otomatis. Tidak ada kerumitan klaim manual.Sustainable Economy:Hadiah berasal dari External Yield (Real Yield), bukan dari inflasi token ZOL. Game bisa berjalan selamanya selama ada likuiditas.8.
